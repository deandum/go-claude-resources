description: Decompose a complex task and delegate to specialist agents

Use the lead agent for complex, multi-concern tasks.

The lead will:
1. Spawn critic to clarify and decompose the task
2. Generate a structured spec file (SPEC-[task-slug].md)
3. Present the spec with execution waves for my approval
4. Execute waves in parallel — one agent per task
5. Verify against the spec's success criteria

The spec IS the prompt for other agents. No interpretation needed.
Do not start execution without my approval of the spec.
